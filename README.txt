TheGameOfTheGenerals
====================

Graduate Project - Converting the board game "The Game Of The Generals" for PC and Android devices.
졸업 프로젝트 - 보드 게임 "더 게임 오브 더 제너럴" 을 PC 및 안드로이드 기기로 이식하기.

This README file will be described about new version each time when commited and pushed to GitHub server.
이 README 파일은 매번 GitHub 서버에 올릴 때마다 해당 새 버전에 대해 기술하게 될 것입니다.

2014. 8. 22. Fri. Version 0.5
  First using GitHub.
  Cannot play this game yet. It isn't completed.
  The part of starting (placing units) has just been completed.
  깃허브를 처음으로 사용하게 되었습니다.
  아직 완료된 프로젝트가 아닙니다. 따라서 플레이가 가능하지는 않습니다.
  처음 시작 부분(말들을 배치하는 부분)까지만 완료된 상태입니다.

2014. 8. 23. Sat. Version 0.7
  It's the first playable version.
  Applied the basic of game logic.
  Can enjoy just once.
  실행 가능한 첫 버전입니다.
  기본 게임 로직을 적용했습니다.
